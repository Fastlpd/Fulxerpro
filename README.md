# Fulxerpro

Top-notch international investment platform

Empowering users to invest, own, and grow assets globally with real-time market data and seamless experience.

## Built With
- Vite + React + TypeScript
- Tailwind CSS v4 (optimized with @tailwindcss/vite plugin)
- Custom components (Hero, Features, Market Data, Invest to Own, Canvas/Scene)

## Quick Start
```bash
# Install dependencies
npm install

# Start development server
npm run dev
